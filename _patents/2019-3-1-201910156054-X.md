摘要
-----
本发明公开了一种优化焦炭生产的方法、装置和设备。该方法包括：根据历史数据构建焦炭质量预测模型，并对焦炭质量预测模型进行训练，得到第一焦炭质量数据，其中，历史数据包括焦炭生产技术参数数据、配合煤数据和焦炭质量数据；根据第一焦炭质量数据和业务需要确定第一焦炭质量数据中的待优化指标数据；采用优化算法对待优化指标数据进行优化，并根据优化结果得到待优化指标数据的建议值；根据建议值调整焦炭生产技术参数数据或配合煤数据，以提高焦炭质量。本发明充分考虑了实际生产中焦炭生产技术参数和配合煤对焦炭质量的影响，因此，优化了焦炭生产。    

Abstract
----
The invention discloses a method, a device and equipment for optimizing coke production. The method comprises the following steps: constructing a coke quality prediction model according to historicaldata, and training the coke quality prediction model to obtain first coke quality data, the historical data comprising coke production technical parameter data, blended coal data and coke quality data; determining to-be-optimized index data in the first coke quality data according to the first coke quality data and business needs; optimizing the to-be-optimized index data by adopting an optimization algorithm, and obtaining a suggested value of the to-be-optimized index data according to an optimization result; and adjusting the coke production technical parameter data or the blended coal dataaccording to the suggested value so as to improve the coke quality. According to the method, the influence of coke production technical parameters and blended coal on the coke quality in actual production is fully considered, so that the coke production is optimized.
