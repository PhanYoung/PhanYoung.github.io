核心方法
-----
使用爬虫获取与目标网页相同路径的其它网页，通过后缀树寻找公共xpath路径，获取domTree的公共节点，进而得到网页模板。  

摘要
-----
本发明公开了一种数据处理方法及装置，方法包括：解析至少一个页面，以分别获取每一个所述页面对应的节点路径信息列表；基于所述节点路径信息列表，对所述至少一个页面中的页面之间进行结构比对，得到相似页面；对所述相似页面的节点路径信息列表设置标签；基于具有相同标签的节点路径信息列表，生成与该标签相匹配的页面抽取模板。    

Abstract
----
The invention discloses a data processing method and device. The method comprises the following steps: analyzing at least one page to obtain a node path information list corresponding to each page; carrying out structural comparison among pages in the at least one page based on the node path information list to obtain similar pages; setting a label on the node path information list of the similarpage; based on the node path information list with the same label, generating a page extraction template matching the label.
