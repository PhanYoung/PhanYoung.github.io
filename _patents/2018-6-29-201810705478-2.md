核心方法
-----
根据元数据变化，获取源数据与目标数据的差异，进行增量更新。  

摘要
-----
本公开提供了一种数据同步方法，该方法包括：获取来自数据源的用于描述内容数据对象的第一数据；获取存储在第一存储空间中用于描述内容数据对象的第二数据，其中，第二数据和已存储元数据对应，第二数据的数据量小于已存储元数据的数据量；比较第一数据和第二数据的差异，得到差异比较结果；以及基于差异比较结果对已存储元数据进行更新。本公开还提供了一种数据同步系统以及一种服务器集群。    

Abstract
----
The invention provides a data synchronization method. The method comprises the steps of acquiring first data for describing a content data object from a data source; acquiring second data which is stored in a first storage space and used for describing the content data object, wherein the second data corresponds to stored metadata, and the data amount of the second data is smaller than that of thestored metadata; comparing the difference between the first data and the second data to obtain a difference comparison result; updating the stored metadata based on the difference comparison result.The invention also provides a data synchronization system and a server cluster.
