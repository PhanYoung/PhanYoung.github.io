核心方法
-----
根据句法关系，将自然语言描述的问题进行结构化解析，并转换为图查询语言。  

摘要
-----
本发明公开的一种数据处理方法及服务器，在获得问题语句后，基于问题语句的句子成分确定问题语句所属的问句类别，并在问题语句属于预定问句类别的情况下，对问题语句进行依存句法分析，得到问题语句的各组成元素间的依存信息，进而基于问题语句的各组成元素间的依存信息以及所述问题语句的句子成分，对问题语句进行结构化解析，得到问题语句的结构化信息。通过利用问题语句的各组成元素间的依存信息以及问题语句的句子成分，对问题语句进行结构化解析，可达到较好的解析效果，进而可基于解析结果准确地识别用户问句意图。    

Abstract
----
A data processing method and a server are disclosed. After the problem statement is obtained, based on the sentence components of the problem statement, the question category to which the problem statement belongs is determined, and if the question statement falls into the predetermined question category, Dependency syntactic analysis is carried out on the problem statement to obtain dependency information among the constituent elements of the problem statement, and then structural analysis is carried out on the problem statement based on the dependency information among the constituent elements of the problem statement and sentence components of the problem statement to obtain structural information of the problem statement. By using the dependency information among the constituent elements of the question statement and the sentence components of the question statement, the problem statement can be structurally parsed, and the user's question intention can be accurately identified based on the parsing results.
