核心方法
-----
将待识别文本中的每个文字依次转换为字向量，通过机器学习获得噪声词的位置标记，和噪声类型。  

摘要
-----
本申请提供了一种文本中噪音词的识别方法、装置、服务器组及存储介质，方法包括：获取待识别文本，将待识别文本中的每个文字依次转换为字向量，获得与待识别文本对应的字向量集合，将与待识别文本对应的字向量集合输入预先建立的噪音词识别模型，得到噪音词识别模型输出的待识别文本中噪音词的识别结果，其中，噪音词识别模型以标注了噪音词的训练文本对应的字向量集合为训练样本进行训练得到。本申请提供的文本中噪音词的识别方法可通过预先建立的噪音词识别模型对待识别文本进行识别，由于噪音词识别模型基于标注了噪音词的训练文本训练得到，因此，通过该噪音词识别模型可从待识别文本中识别噪音词。    

Abstract
----
The invention provides an identification method, device, server group and storage medium for noise words in a text. The method comprises the steps that a to-be-identified text is acquired, each word in the to-be-identified text is converted into a word vector in turn, a word vector set corresponding to the to-be-identified text is acquired, the word vector set corresponding to the to-be-identifiedtext is input in a preconstructed noise word identification model, an identification result of the to-be-identified text output by the noise word identification model is obtained, wherein a word vector set corresponding to a training text with annotating the noise words is taken as a training sample for training to obtain the noise word identification model. According to the identification methodfor the noise words in the text, the to-be-identified text can be identified through a preconstructed noise word identification model, because the noise word identification model is obtained based onthe training text with annotating the noise words, the noise words can be identified in the to-be-identified text through the noise word identification model.
