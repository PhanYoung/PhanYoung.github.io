核心方法
-----
使用词向量和深度学习进行实体对齐。  

摘要
-----
本发明公开了一种实体匹配方法、装置及电子设备，该方法包括：获取待匹配的实体名称；根据该实体名称所包含的字符串，确定用于表征该实体名称所表达的语义的第一编码向量；分别计算该实体名称的第一编码向量与标准名称库中各个标准实体名称的第二编码向量之间的距离；从该标准名称库的多个标准实体名称中，选取出对应的该距离较小的至少一个候选实体名称；根据该实体名称的第一编码向量和该候选实体名称的第二编码向量，确定该候选实体名称与该实体名称在多个预设维度上的特征匹配度；将该特征匹配度符合预设要求的候选实体名称确定为该实体名称对应的标准实体名称。本申请可以提高实体匹配的精准度，并减少人力资源消耗。    

Abstract
----
The invention discloses an entity matching method and device, and an electronic device. The method comprises the steps of obtaining an entity name to be matched; determining a first coding vector usedfor representing semantics expressed by the entity name according to the character string contained in the entity name; respectively calculating the distance between the first coding vector of the entity name and the second coding vector of each standard entity name in the standard name library; selecting at least one corresponding candidate entity name with a smaller distance from a plurality ofstandard entity names in the standard name library; determining feature matching degrees of the candidate entity names and the entity names in a plurality of preset dimensions according to the firstcoding vectors of the entity names and the second coding vectors of the candidate entity names; and determining the candidate entity names of which the feature matching degrees meet preset requirements as standard entity names corresponding to the entity names. According to the invention, the accuracy of entity matching can be improved, and the manpower resource consumption is reduced.
