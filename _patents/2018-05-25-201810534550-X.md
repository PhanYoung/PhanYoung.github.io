核心方法
-----
根据一系统任务的依赖关系和先后顺序，把任务流解析为有向无环图（DAG），基于DAG调度任务。  

摘要
-----
本公开提供了一种用于数据分析流程调度的方法，包括：根据单个业务数据分析流程之间的依赖关系，获取多业务的分析有向流程；根据预设的调度规则，或者预设的调度规则和预设的依赖规则对所述分析有向流程进行调度。本公开还提供了一种用于数据分析流程调度的装置以及一种服务器集群。    

Abstract
----
The invention provides a method used for data analysis process scheduling. The method comprises the following steps that: according to a dependency relationship in a single business data analysis process, obtaining the analysis directed process of multiple businesses; and according to a preset scheduling rule or a preset scheduling rule and a preset dependency rule, scheduling the analysis directed process. The invention also provides a device used for data analysis process scheduling, and a server cluster.
