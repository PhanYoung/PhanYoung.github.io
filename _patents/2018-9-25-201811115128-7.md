核心方法
-----
使用无限长度的循环神经网络预测流程制造设备的目标指标变化。  

摘要
-----
本发明提供的一种数据处理方法、服务器及电子设备，可利用循环神经网络训练一预测模型，该预测模型用于对预定设备的目标指标进行预测。在训练预测模型时，除了以训练样本为模型输入，还引入了待预测指标所对应的所述预定设备的设备状态信息，并具体以第一训练样本的上一样本在训练过程中输出的第一设备状态信息作为第一训练样本在进行训练时的状态输入信息，这就使得设备状态信息能够按各样本的训练次序在各样本的训练过程中进行传递，相对应地能够使得所得的预测模型达到对设备状态信息的长时记忆效果，后续在利用该预测模型进行预测时，能够结合预测样本的数据及记忆的设备状态信息对目标指标进行预测，从而可使得预测结果具备较高的准确率。    

Abstract
----
The present invention provides a data processing method, a server and an electronic device, which can train a prediction model by using a circulating neural network, and the prediction model is used for predicting a target index of a predetermined device. In a process of training the prediction model, In addition to taking training samples as model input, equipment state information of the predetermined equipment corresponding to an index to be predicted is also introduced, and the first equipment state information outputted from the upper sample of the first training sample in the training process is taken as the state input information of the first training sample at the time of training, this enables the equipment state information to be transmitted in the training process of each sample according to the training order of each sample, Correspondingly, the obtained prediction model can achieve the long-term memory effect of the equipment state information, and when the prediction model is used for subsequent prediction, the target index can be predicted by combining the data of the prediction sample and the memorized equipment state information, so that the prediction result canhave a higher accuracy rate.
